# Smart Business Management and Tracking System

A business platform for sales, inventory, employee management, receipt processing, and analytics.

## Start Here

- Developer setup and run instructions: [DEVELOPER_GUIDE.md](DEVELOPER_GUIDE.md)
- API reference: [docs/API.md](docs/API.md)

## Core Tech Stack

- Frontend: React (Vite)
- Backend: FastAPI (Python)
- Databases: PostgreSQL and MongoDB
- Container workflow: Docker Compose
